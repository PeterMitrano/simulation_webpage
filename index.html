About

FRCSim is a project started in 2014, with the goal of providing simulation tool that would allow teams to test and prototype faster. We know what it's like to be writing and testing brand new code two days before you bag the robot. So, we thought about how to help students test their code before the robot is done. Gazebo is a multi-robot simulator used heavily in research and in industry. FRCSim allows your WPILib C++ or Java robot code to control your robot in gazebo, just like it controls your real robot!
